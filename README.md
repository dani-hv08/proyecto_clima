# proyecto_clima
Este proyecto permite consultar el clima de diferentes ciudades utilizando una API de datos meteorológicos.  
Fue desarrollado como práctica para el curso de **Programación I**.

---

## Funcionalidades
- Consulta el clima actual por nombre de ciudad.
- Muestra temperatura, humedad, presión y estado del cielo.
- Integra datos de una API pública de clima.
- Interfaz sencilla en consola.

---

## Tecnologías utilizadas
- **Python 3**
- **Requests** (para conectarse a la API)
- **JSON** (para manejar respuestas)
- **Virtualenv** (entorno virtual)
